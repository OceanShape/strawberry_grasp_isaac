# Strawberry Grasp Robot - Isaac Sim Asset

이 폴더(`robot`)는 아이작 심(Isaac Sim)에서 딸기 수확 로봇 환경을 구성하기 위한 **독립적인(Self-contained) 에셋 폴더**입니다.
이 폴더 하나만 있으면 어떤 프로젝트나 컴퓨터에서도 로봇 환경을 그대로 불러올 수 있습니다.

## 🚀 사용 방법

1. **폴더 복사**: 
   이 `robot` 폴더 전체를 새로운 아이작 심 프로젝트(또는 원하는 경로)로 복사합니다.
   
2. **에셋 불러오기**:
   아이작 심(Isaac Sim)을 실행한 후 하단 Content 창을 통해 복사한 `robot` 폴더로 이동합니다.
   목적에 따라 다음 파일 중 하나를 드래그 앤 드롭(또는 Import)하여 사용하세요.

   * 🍓 **전체 환경이 필요할 때 (`strawberry_grasp_robot.usd`)**
     로봇 팔, 테이블, 그리고 목적물인 딸기(익은 딸기, 안 익은 딸기)가 모두 포함된 최상위 씬 파일입니다.
     - **참고**: 모든 텍스처와 참조가 상대 좌표로 수정되어 있어, `robot` 폴더 구조만 유지된다면 텍스처 깨짐 없이 동작합니다.

   * 🤖 **로봇 단일 모델만 필요할 때 (`robot.usd` 또는 `robot_recent.usd`)**
     배경이나 딸기 없이, Doosan 로봇 팔과 커스텀 그리퍼, 센서만 결합된 에셋입니다.

   * 🍓 **딸기 단일 모델만 필요할 때 (`strawberry_ripe.usd` / `strawberry_unripe.usd`)**
     물리 속성과 텍스처가 적용된 개별 딸기 에셋입니다.

3. **레이아웃 불러오기 (선택 사항)**:
   함께 동봉된 `strawberry_grasp.json` 파일은 아이작 심의 UI/창 배치 레이아웃 설정 파일입니다.
   필요한 경우 아이작 심 설정(Window -> Layout -> Import)에서 이 파일을 불러와서 작업 환경을 복원할 수 있습니다.

## 📂 폴더 구조 설명
* `configuration/`: 로봇 팔(base, physics, sensor 등)을 구성하는 하위 usd 파일들이 포함되어 있습니다.
* `textures/`: 전체 씬과 딸기에 사용되는 텍스처 이미지 및 HDRI 파일이 모여있습니다. 이 폴더 덕분에 외부 절대 경로 의존성 없이 에셋이 독립적으로 동작합니다.
* `backup/`: 이전 작업 내역 백업본 파일들이 있습니다.

---
**주의**: 이 폴더 내부 파일들의 이름을 바꾸거나 구조를 흩어놓으면 내부 상대 경로 연결이 끊어져 텍스처나 메쉬가 깨질 수 있습니다. 가급적 폴더 구조를 그대로 유지해 주세요.
